# Fanbasemarket Site

## System Requirements
* Python 3.7
* npm
* Any version of MySQL

## Setup Instructions
### Flask Setup
* `pip install -r requirements.txt`
* `python setup.py install`
* `pip install -e .`
### React Setup
* `cd frontend`
* `npm install`

## Running Instructions
### For API
* `flask run`
### For Frontend
* `cd frontend`
* `npm start`
